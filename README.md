# ChatSpace
ChatSpace - A Realtime Chat App created using Nodejs &amp; Socket.io

## Demo

Click [here](https://chatspace-somsubhra.herokuapp.com/) for demo.

## Technologies:

-   NodeJS
-   ExpressJS
-   Socket.io


### Version

1.0.0

## Usage

### Installation

Install the dependencies

```sh
$ npm install
```

Run app

```sh
$ npm start
```
